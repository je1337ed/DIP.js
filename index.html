Announcing first beta of DIP.js

DIP.js is a digital signal processing library for images, written in javascript. Right now it's in beta, as the code needs some serious refactoring and hasn't been tested outside of my demos. The codebase contains abstractions for managing pixel data from images loaded in canvas, spatial transforms, frequency transforms, and showing isometric projection of various data.
